What changes from the current dashboard

  • KPI cards — bigger numbers (32px bold), icon badges, mini progress bars, ▲ trend on "This Month"
  • Export Value card — stacked per-currency (USD dominant, EUR/GBP smaller below)
  • Drafts card — amber border + ⚡ alert to prompt action
  • Invoice table — proper 4-col header, amounts with currency prefix, rounded pill badges
  • Chart — SVG with horizontal grid lines, current month solid indigo, rest light indigo
  • Destinations — indigo gradient bars (darkest = most)
  • Activity feed — dot + connector timeline, shows invoice + PO events from existing created_at data. Indigo dot = invoice created, green = final, amber = PO

Zero DB migrations. Activity uses invoices.created_at + purchase_orders.created_at already in the schema.

Note on "finalized" in activity: since there is no status_changed_at column, "finalized" events show final invoices ordered by updated_at (already on the table).
